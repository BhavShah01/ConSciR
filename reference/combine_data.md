# Combine TRH data from a list

Combines a list of parsed logfiles and returns a

## Usage

``` r
combine_data(datalist)
```

## Arguments

- datalist:

  A list of parsed dataframes

## Value

A dataframe containing all timestamped rows in \`datalist\`
